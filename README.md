# 🐍 Python Mini Projects Collection

This repository contains a collection of beginner-friendly Python mini projects. These projects helped me practice core programming concepts like loops, conditions, functions, and basic logic building.

---

## 📌 Projects Included

### 1. 💰 Millionaire Quiz Game (KBC Style)

* A quiz-based game inspired by *Kaun Banega Crorepati (KBC)*
* The user is asked multiple-choice questions
* Correct answers increase the prize money 💵
* Wrong answer ends the game

👉 **Concepts Used:**
`if-else`, `lists`, `input/output`, `loops`

---

### 2. 🔢 Number Missing Game

* A number guessing game
* The system generates a random number
* User guesses the number with hints like:

  * "Too High ⬆️"
  * "Too Low ⬇️"

👉 **Concepts Used:**
`random module`, `loops`, `conditions`

---

### 3. 📱 Phone Book Mini Project

* Simple contact management system
* Add, view, and search contacts

👉 **Concepts Used:**
`dictionary`, `functions`, `user input`

---

### 4. 🔐 Password Generator

* Two versions:

  * With limits
  * Without limits
* Generates secure random passwords

👉 **Concepts Used:**
`random`, `string`, `loops`

---

### 5. 🧾 QR Code Generator

* Converts text or links into QR codes
* Useful for sharing URLs quickly

👉 **Concepts Used:**
`qrcode library`, `file handling`

---

### 6. 🎲 Roll and Dice Game

* Simulates dice rolling
* Generates random numbers like real dice

👉 **Concepts Used:**
`random module`

---

### 7. 🧮 Simple Calculator

* Performs basic arithmetic operations:

  * Addition ➕
  * Subtraction ➖
  * Multiplication ✖️
  * Division ➗

👉 **Concepts Used:**
`functions`, `conditions`

---

### 8. 🏦 Simple Bank Application

* Basic banking operations:

  * Deposit 💰
  * Withdraw 💸
  * Check Balance

👉 **Concepts Used:**
`functions`, `loops`, `conditions`

## 🚀 How to Run the Projects

1. Clone this repository:

```bash
git clone https://github.com/your-username/Python_mini_Projects.git
```

2. Navigate to the project folder:

```bash
cd Python_mini_Projects
```

3. Run any file:

```bash
python filename.py
```
